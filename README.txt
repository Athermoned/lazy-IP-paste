# lazy-IP-paste
Программа для составления множества команд с IP адресами. 
Вы вводите начальные и конечные октеты IP адреса, вводите комманду, которая должна работать с вашими IP адресами и программа сохраняет комманды в файл "ip.txt"
Пример: 
Вводите первые два октета. 192 и 168. Дальше вводите 3-й и 4-й октет, напрмер 1 и 1, потом вводите конечные 3-й и 4-й октет. Пускай будет 5 и 5.
Дальше вводим комманду, например ping, затем добавляем {ip}, которая добавляет наши IP адреса.
